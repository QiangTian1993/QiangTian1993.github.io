                                    # 常用 HTML 标签

                                    # div

                                    块级元素，默认独占一行，可自定义宽高

                                    div元素 (或 HTML 文档分区元素) 是一个通用型的流内容容器，它在语义上不代表任何特定类型的内容，它可以被用来对其它元素进行分组，一般用于样式化相关的需求（使用 class 或 id 特性) 或者对具有相同特性的一组元素进行分组 (比如 lang)，它应该在没有任何其它语义元素可用时才使用

                                    <div></div>
                                    1

                                    # span

                                    行内元素，默认从左到右排列，不可自定义宽高，它的宽高受它内部元素的大小决定

                                    span 元素是短语内容的通用行内容器，并没有任何特殊语义。可以使用它来编组元素以达到某种样式意图（通过使用类或者 Id 属性），或者这些元素有着共同的属性，比如 lang。应该在没有其他合适的语义元素时才使用它。span与div元素很相似，但div是一个 块元素 而 span 则是 行内元素

                                    <span></span>
                                    1

                                    # img

                                    img 元素将一份图像嵌入文档。

                                    img 可替换元素，表示一个图片，宽高受源图片大小的影响，可以自定义宽高

                                    <img src="图片地址" alt="图片未显示时替换文本" onerror="this.src=../../assets/img/timg.jpg"></img>
                                    1

                                    # a

                                    行内元素

                                    a 元素（或称锚元素）可以创建通向其他网页、文件、同一页面内的位置、电子邮件地址或任何其他 URL 的超链接。

                                    <!-- 链接到外部地址 -->
<a herf="https://www.baidu.com/">百度</a>
<!-- 链接到本页的某个部分 -->
<a herf="#xxx">百度</a>
<!-- 建立伪协议 -->
<a href="javascript: alert(1);"> <a href="javascript:;"></a></a>

                                    a 连接在跳转的时候还可以自定义其跳转方式

                                    <!--新建窗口打开_blank-->
<a herf="https://www.baidu.com/" "_blank>百度</a>
<!--当前窗口打开_self-->
<a herf="https://www.baidu.com/" "_self">百度</a>
<!--父级窗口打开_parent-->
<a herf="https://www.baidu.com/" "_parent>百度</a>
<!--顶级窗口打开_top-->
<a herf="https://www.baidu.com/" "_top>百度</a>

                                    # input

                                    input 元素用于为基于 Web 的表单创建交互式控件，以便接受来自用户的数据; 可以使用各种类型的输入数据和控件小部件

                                    # 输入框

                                    type 属性为 text 时是最普通的文本输入框

                                    <!--如果未定义input的type类型默认为text-->
<input type="text" />
                                    1
                                    2

                                    type 属性为 number 时只能输入数字，min 和 max 能控制数字的区间

                                    <input type="number" min="1" max="10"></input>
                                    1

                                    type 属性为 password 时可隐藏输入字段

                                    <input type="password"></input>
                                    1

                                    # 按钮

                                    type 属性为 button 时可定义一个按钮

                                    <!--使用value定义button内文案 -->
<input type="button" value="button" />
                                    1
                                    2

                                    type 属性为 submit 时可定义一个提交按钮

                                    <input type="submit" />
                                    1

                                    # 复选框

                                    type 属性为 checkbox 时可变成一个复选框

                                    <!--复选框，必须定义value值否则无法提交 -->
<input type="checkbox" value="woman" >女生</input>
<!--使用checked可定义控件是否默认被选择-->
<input type="checkbox" value="woman" checked >女生</input>
<!--使用disabled可控制是否可选 -->
<input type="checkbox" value="woman" disabled>女生</input>
                                    1
                                    2
                                    3
                                    4
                                    5
                                    6

                                    如果想第一组复选框可使用 name 属性

                                    <input type="checkbox" value="0" name="fruit">香蕉</input>
<input type="checkbox" value="1" name="fruit">苹果</input>
                                    1
                                    2

                                    # 单选框

                                    type 属性为 radio 时可变成一个单选框

                                    <input type="radio" value="0" name="fruit">香蕉</input>
<input type="radio" value="1" name="fruit">苹果</input>
                                    1
                                    2

                                    # 日历控件

                                    type 属性为 date 时可定义一个日历控件

                                    <input type="date" />
                                    1

                                    # 带时间的日历控件

                                    type 属性为 datetime-local 时可定义一个带时间的日历控件

                                    <input type="datetime-local" />
                                    1

                                    # 只包含年月的日历控件

                                    <input type="month" />
                                    1

                                    # 时间控件

                                    <input type="time" />
                                    1

                                    # 已周为周期的日历控件

                                    <input type="week" />
                                    1

                                    # 上传文件

                                    type 属性为 file 时可定义一个上传控件

                                    <input type="file" />
<!--accept定义选择文件类型-->
<input type="file" accept="image/png" />
                                    1
                                    2
                                    3

                                    # 滑块

                                    type 属性为 range 可定义一个滑块

                                    <input type="range" min="0" max="100" value="0" />
                                    1

                                    # form

                                    form 元素 表示了文档中的一个区域，这个区域包含有交互控制元件，用来向 web 服务器提交信息

                                    通常 input 组件都会放在 form 标签中

                                    <!--使用action定义处理请求的URL，使用method定义发起请求的类型-->
<form action="" method="POST">
	<input type="text" name="userName" />
	<input type="password" name="password" />
	<input type="submit" value="提交" />
</form>

    <label>密码：<input type="password" name="password"><label>
    <input type="submit" value="提交">
</form>

                                    # table

                                    table元素表示表格

	<!--表头-->
	<thead>
		<tr>
			<th>标题1</th>
			<th>标题2</th>
		</tr>
	</thead>
	<!--表内容-->
	<tbody>
		<tr>
			<td>内容1</td>
			<td>内容2</td>
		</tr>
	</tbody>
	<!--表尾-->
	<tfoot>
		<tr>
			<td>内容1</td>
			<td>内容2</td>
		</tr>
	</tfoot>
</table>

                                    # header

                                    header 标签用来表示文档的头部

                                    # main

                                    main 标签表示文档内的主题内容

                                    # footer

                                    footer 标签用来表示文档的页脚

                                    # aside

                                    aside 标签用来表示侧边栏和标注框

                                    # section

                                    section 标签表示独立文档的部分，通常都会包含一个标题

                                    # article

                                    article 标签表示独立于网站或文档的部分，通常会用来表示一些可复用的元素，如在发布中，它可能是论坛帖子、杂志或新闻文章、博客、用户提交的评论、交互式组件

                                    # figure

                                    这个标签经常是在主文中引用的图片，插图，表格，代码段等等

                                    <details>
	<summary>详情</summary>
	<span>内容</span>
</details>
