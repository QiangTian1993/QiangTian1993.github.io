    # 计算属性与监听属性

    # 计算属性

    计算属性写在 Vue 实例的 computed 属性下

    所以需要计算的属性，都已函数的形式写在 computed 中

    计算属性的默认用法是 getter 函数,属性后直接跟函数

    get 获取 data 里的数据

    set 修改或添加属性

    如果你给属性设置值的时候，就会调用 setter 方法

    // 不写get默认是get
computed: {
    fullName:{
        get:function(): {

        }
        set:function():{

        }
    }
}

    # 监听属性

    watch 监听属性，当属性发生变化是执行

    data: {
    firstName: '徐坤',
    lastName: '蔡',
    fullName: '蔡徐坤'
}
watch: {
    // 当 firstName 发生改变时开始执行函数
    firstName: function(){
        return this.fullName = this.lastName +this.firstName
    }
    // 当 lastName 发生改变时开始执行函数
    lastName: function(){
        return this.fullName = this.lastName +this.firstName
    }
}

    监听函数还可以接收两个参数 val oldVal > val 表示新的值 oldVal 表示修改之前的值

    # 监听属性的 handler 方法 和 immediate 属性

    我们知道 watch 默认是属性变化时才会执行，如果想组件绑定时就执行，这时就需要用到 immediate 属性和 handler 函数了

    watch: {
    firstName: {
        handler(val,olVal){
        },
        immediate: true // 将 immediate 属性置为 true 这样组件绑定的时候就会执行 handler 函数中的方法
    }
}

    # deep 属性

    deep 属性深度监听，如果想监听对象里面的属性就可以将 deep 置为 true

    data() {
    return{
        data: {
            name: 'ian'
        }
    }
},

watch: {
    data: {
        handler(newVal,oldVal) {
            if(oldVale.name != newVal.name){
                console.log(newVal.name)
            }
        },
        deep: true
    }
}

    return{
        data: {
            todo: {
				id: 0,

			}

    }
},

watch: {
    getTodo: {
        handler(newVal,oldVal) {
			console.log(newVal,oldVal)
        },
        deep: true
    }
}

computed: {
	getTodo() {
		return this.todo
	}
