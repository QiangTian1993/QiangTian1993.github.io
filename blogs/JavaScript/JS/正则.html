# 正则

# 正则的实例方法

# RegExp.prototype.test()

正则实例对象的 test 方法返回一个布尔值，表示当前模式是否能匹配参数字符串。

var reg = /W/g

var str1 = 'Hello World'
var str2 = '你好世界'

reg.test(str1) // true
reg.test(str2) // false

# RegExp.prototype.exec()

正则实例对象的 exec 方法，用来返回匹配结果。如果发现匹配，就返回一个数组，成员是匹配成功的子字符串，否则返回 null。

var reg = /o/g

var str1 = 'Hello World'
var str2 = '你好世界'

reg.exec(str1) //  ["o", index: 4, input: "Hello World", groups: undefined]
reg.exec(str1) // ["o", index: 7, input: "Hello World", groups: undefined]
reg.exec(str1) // null
reg.exec(str2) // null

# 字符串的实例方法

# String.prototype.match()

match 方法返回一个数组，是所有匹配的所有字符串。如果没有匹配的就返回 null

var str = 'Hello World'

str.match(/o/g) // ['o','o']

# String.prototype.search()

search 方法返回最后一个匹配的的索引, 如果没有匹配到就会返回 -1

var str = 'Hello World'

str.search(/o/) // 4
str.search(/f/) // -1

# String.prototype.replace()

字符串对象的 replace 方法可以替换匹配的值。它接受两个参数，第一个是正则表达式，表示搜索模式，第二个是替换的内容。

var str = 'Hello World'

str.replace(/o/g, 'O') // 'HellO WOrld'

# String.prototype.split()

字符串对象的 split 方法按照正则规则分割字符串，返回一个由分割后的各个部分组成的数组。

var str = 'a , b, c ,d'

str.split(/ , /) // [a, b, c, d]
1
2
3

/d/ 会在 'Hello World' 中匹配 'd'

/o/g 会在 'Hello World' 中匹配所有的 'o'

/^H/会在 'Hello World HHHHH' 匹配以 'H' 开头的

/H\$/ 会在 'Hello World HHHHH' 匹配以 'H' 结尾的

/llo/ 会在 'Hello World' 匹配 'llo' 或 ’lo‘

/ol?/ 不在乎顺序，在 'Hello World'中匹配 'lo' 或 'ol'

/A|B/匹配 A 或者 B \s 空格 \w 获取首字母

/H(\w)/ 匹配 H 和其后面的一个字符

/([^\?]+)\$/ 匹配 ？ 后面的所有内容

/H(/S)/ 匹配 H 包括 H 后面的所有内容

/^H.d\$/ 匹配已 H 开头并以 d 结尾

i 忽略大小写 g 匹配所有符合条件的

/^[A-Za-z0-9\u4e00-\u9fa5]+$ // 不允许输入空格等特殊符号
