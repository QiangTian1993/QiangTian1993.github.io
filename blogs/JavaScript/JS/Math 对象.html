    Math 对象

    IAN
    2020-11-03

    Math 对象平常开发用的频率比较少，每次用完之后就忘 🤣 ，今天干脆做个总结加深下记忆

    # Math 对象

    Math javaScript 中的内置对象，它拥有一些数学常数和数学方法。Math 用于 Number 类型，不支持 BigInt

    # Math 对象的属性

    Math 的属性这里就不多说了，主要是一些数学常数，详情请查看 MDN

    • Math.E 欧拉常数，也是自然对数的底数，约等于 2.718。

    • Math.E 2 的自然对数，约等于 0.693。

    • Math.LN10 10 的自然对数，约等于 2.303。

    • Math.LOG2E 以 2 为底的 E 的对数，约等于 1.443。

    • Math.LOG10E 以 10 为底的 E 的对数，约等于 0.434。

    • Math.PI 圆周率，一个圆的周长和直径之比，约等于 3.14159。

    • Math.SQRT1_2 二分之一 ½ 的平方根，同时也是 2 的平方根的倒数，约等于 0.707。

    • Math.SQRT2 2 的平方根，约等于 1.414。

    # Math 的方法

    # Math.min 和 Math.max

    Math.min 和 Math.max 用来返回一组数中的最小值和最大值

    Math.min(1, 2, 3, 4) // 1
Math.min(1, 2, 3, 4) // 4
    1
    2

    它们多用于求一个数组中的最大值或最小值，假设有以下数组：

    var arr = Array(50)
	.fill()
	.map(v => Math.ceil(Math.random() * 100))
    1
    2
    3

    我们可以结合展示操作进行求值

    Math.min(...arr)
Math.max(...arr)
    1
    2

    还可以函数的 apply 方法以数组进行传参的特性

    Math.min.apply(Math, arr)
Math.max.apply(Math, arr)
    1
    2

    # 舍入方法

    1. Math.ceil 向上取整
    Math.ceil(2.1) // 3
    1
    1. Math.floor 向下取整
    Math.floor(2.9) // 2
    1
    1. Math.floor 正常四舍五入
    Math.round(2.3) // 2
Math.round(2.6) // 3
    1
    2

    # Math.random

    Math.random 方法返回一个大于 0 小于 1 的随机数。 它的应用比如生成一个长度为 10 的数组，数组中的每一项大于 1 小于 100

    var arr = Array(10)
	.fill()
	.map(v => Math.ceil(Math.random() * 100 + 1))
    1
    2
    3

    # Math.abs

    Math.abs 取绝对值

    Math.abs(-100) // 100
    1

    # Math.pow

    Math.pow(x, y) x 的 y 次幂

    Math.pow(2, -1) // 0.5
Math.pow(2, 1) // 2
Math.pow(2, 2) // 4
    1
    2
    3
